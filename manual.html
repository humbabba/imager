1. Overview

Sublogical Endeavors Imager is a browser-based image processing tool. All processing happens locally in your browser — no images are uploaded to any server.

Key capabilities:

  • Resize, crop, and reformat images
  • Apply aspect ratios with crop or overlay modes
  • Add text overlays with full styling control
  • Process multiple images in batch
  • Save and share your settings via URL

Supported input formats: JPEG, PNG, GIF, WebP, and BMP.

2. Getting Started

The basic workflow is:

  1. Upload — Select one or more images using the file picker.
  2. Configure — Set your desired output dimensions, aspect ratio, resize mode, and format.
  3. Process — Click the Process button. The source and output previews appear on either side of the control panel.
  4. Text (optional) — Switch to the Text Overlay tab to add styled text to your image.
  5. Download — Download the final result as a single image or a ZIP archive for batch jobs.

3. Uploading Images

Click the Select File(s) button at the top of the Process panel. You can select a single image or multiple images at once using your operating system's file picker.

After selecting files, the Process button updates to show the count (e.g., "Process 5 Images"). For a single image it reads "Process Image".

The Source preview panel appears on the left showing the first selected image along with its filename, dimensions, and file size.

4. Output Filename

The Output Filename field lets you set a custom name for your processed image. If left blank, the original filename is used as the base.

Two checkboxes modify the naming behavior:

  • Keep name — When checked, the custom filename you entered persists even when you upload new images. When unchecked, the filename field resets to blank on each new upload.
  • Timestamp — Enabled by default. Appends a timestamp in the format YYYY-MM-DD-HH-MM-SS to the output filename, ensuring unique names and preventing overwrites.

Example output: my-photo - 2026-02-21-14-30-00.jpg

5. Dimensions

Set maximum output dimensions in pixels:

  • Max width — The output image will not exceed this width. Leave blank for no width constraint.
  • Max height — The output image will not exceed this height. Leave blank for no height constraint.

When both are set, the image is scaled to fit within those bounds while maintaining its proportions (unless overridden by an aspect ratio).

When neither is set, the image retains its original dimensions (cropped or padded to fit the aspect ratio if one is selected).

6. Aspect Ratio

Choose a preset ratio from the dropdown or enter a custom one:

  • None — Keep the image's natural proportions
  • 1:1 — Square
  • 4:3 — Classic photo / monitor
  • 3:2 — Standard DSLR photo
  • 16:9 — Widescreen / video
  • 9:16 — Vertical / mobile stories
  • 3:4 — Portrait
  • 2:3 — Tall portrait

For custom ratios, enter width and height values in the W and H fields. Selecting a custom ratio clears the dropdown and vice versa.

When an aspect ratio is selected, the resize mode (Crop or Overlay) determines how the image is adapted to fit.

7. Resize Mode

When an aspect ratio is selected, you choose how the image is resized to match:

  • Crop — Fills the target area completely by scaling the image up and trimming the excess. No empty space. See Crop Mode.
  • Overlay — Fits the entire image inside the target area with a decorative background filling the remaining space. See Overlay Mode.

8. Crop Mode

When Crop is selected, a Crop position dropdown appears with options that vary based on the aspect ratio and source image orientation:

  • Center crop — Always available. Crops equally from both sides.
  • Top crop — Keeps the top of the image, trims the bottom. Available when the image is taller than the target ratio.
  • Bottom crop — Keeps the bottom, trims the top.
  • Left crop — Keeps the left side, trims the right. Available when the image is wider than the target ratio.
  • Right crop — Keeps the right side, trims the left.

Crop mode is ideal when you want the output to be completely filled with image content and don't mind losing some of the edges.

9. Overlay Mode

When Overlay is selected, additional background options appear:

  • Margin — Space between the image and the canvas edge, as a percentage (0–50%). Higher values make the centered image smaller within the frame.
  • Blur — When checked (default), the background is a blurred, scaled-up version of the source image, creating a pleasing "frosted glass" effect.
  • Solid color — When Blur is unchecked, a color picker appears. Choose any solid color for the background.

Overlay mode preserves the entire image and is ideal for fitting photos into standardized dimensions (e.g., social media posts) without losing any content.

10. Drop Shadow (Overlay Mode)

Available only in Overlay mode. Adds a shadow behind the centered image to give it depth against the background.

  • Enable checkbox — Toggle the shadow on or off (enabled by default).
  • Color — Click the color swatch to pick a shadow color.
  • Opacity — 0% (invisible) to 100% (fully opaque). Default: 80%.
  • X offset — Horizontal shift in pixels (-50 to 50). Positive values move the shadow right.
  • Y offset — Vertical shift in pixels (-50 to 50). Positive values move the shadow down.
  • Blur — Shadow blur radius in pixels (0–100). Higher values create a softer, more diffused shadow.

11. Output Format

Controls the file format and compression of the output image:

  • Force JPG output — When checked (default), all images are converted to JPEG regardless of input format. This typically results in smaller file sizes. When unchecked, the output matches the input format (PNG stays PNG, etc.).
  • Quality — JPEG compression quality from 1% (smallest file, most artifacts) to 100% (largest file, best quality). Default: 80%. Only visible when the output will be JPEG.

12. Processing & Download

Click the Process Image button to apply your settings. The button is disabled until at least one image is selected.

After processing, two preview panels appear:

  • Source panel (left) — Shows the original image with its filename, dimensions, and file size.
  • Output panel (right) — Shows the processed result with updated info. Contains:
    • Download — Saves the image to your computer.
    • Preview (◊) — Opens the full-size image in a new browser window.

The Text Overlay tab also becomes available after processing, indicated by the tab bar appearing above the panel.

13. Text Overlay

The Text Overlay view lets you add styled text to your processed image. Switch to it using the Text Overlay tab.

The layout has two main areas:

  • Left panel — Text items list and styling controls.
  • Center canvas — Live preview of the image with all text items. Click anywhere on the canvas to add a new text item at that position.

Managing Text Items

  • Add — Click + Add Text or click directly on the canvas.
  • Select — Click a text item in the list to select it for editing.
  • Move — Drag a selected text item on the canvas to reposition it.
  • Delete — Click the × button on a text item or use the Delete button in the style controls.

An output bar below the canvas shows the file info and provides Download and Preview buttons for the final composited image. You can also change the output filename here.

14. Text Styling

When a text item is selected, the style controls become available:

  • Text — Edit the text content. Changes appear live on the canvas.
  • Font — Choose from 45+ fonts organized into five categories:
    • Sans-Serif — Arial, Archivo, Exo 2, Inter, Lato, Montserrat, Nunito, Open Sans, Poppins, Raleway, Roboto, Ubuntu, Verdana, Work Sans
    • Serif — Bitter, Cinzel, Crimson Text, Georgia, Lora, Merriweather, Playfair Display, PT Serif, Source Serif Pro, Times New Roman
    • Display — Anton, Bangers, Bebas Neue, Fredoka, Lobster, Orbitron, Oswald, Permanent Marker, Righteous, Staatliches, Teko
    • Handwriting — Caveat, Comic Neue, Courgette, Dancing Script, Great Vibes, Indie Flower, Pacifico, Satisfy
    • Monospace — Courier New, Courier Prime, Roboto Mono, Source Code Pro
  • Size — Font size in pixels (8–500).
  • Color — Click the color swatch to pick any text color.
  • Style toggles:
    • B — Bold
    • I — Italic
    • U — Underline
    • AA — Uppercase (converts all text to capitals)
  • Opacity — Slider from 0% (invisible) to 100% (fully opaque).

Positioning

  • ↔ H — Centers the text item horizontally on the canvas.
  • ↕ V — Centers the text item vertically on the canvas.

Text positions are stored as percentages, so they scale correctly if the image dimensions change.

15. Text Effects

Outline

Adds a stroke around the text characters.

  • Enable — Check the box to turn on the outline.
  • Color — Pick the outline color.
  • Opacity — 0–100%. Controls outline transparency independently from the text.

Drop Shadow

Adds a shadow behind the text for depth and readability.

  • Enable — Check the box to turn on the shadow.
  • Color — Pick the shadow color.
  • Opacity — 0–100%.
  • X / Y — Shadow offset in pixels (-50 to 50).
  • B (Blur) — Blur radius (0–100px).

16. Text Background

Adds a colored rectangle behind the text, useful for ensuring readability over busy images.

  • Enable — Check the box to show the background.
  • Color — Background fill color.
  • Opacity — 0–100%. Semi-transparent backgrounds let the image show through.
  • Pad (Padding) — Space between the text and the edge of the background rectangle (0–200px).
  • W (Width) — Background width as a percentage of the canvas (0–100%). When set to 0, the background auto-sizes to fit the text.
  • Off (Offset X) — Horizontal offset of the background (-500 to 500px). Shifts the background left or right relative to the text.
  • Slant L / Slant R — Skew the left and right edges of the background independently (-100 to 100). Creates parallelogram or trapezoid shapes for dynamic, angled banner effects.

17. Batch Processing

Select multiple files to enable batch mode. All images are processed sequentially with the same settings.

  • The Process button shows the total count (e.g., Process 10 Images).
  • A progress indicator appears during processing ("Processing 3 of 10...").
  • Text overlays, if configured, are applied to every image in the batch.
  • The Download button changes to Download ZIP, packaging all processed images into a single archive.
  • Filenames are automatically deduplicated within the ZIP to prevent conflicts.

18. Settings & Sharing

Automatic Persistence

All settings are automatically saved to your browser's local storage as you change them. When you return to Imager, your last-used settings are restored automatically.

Sharing via URL

Click the Copy Settings Link button in the footer to copy a URL that encodes all your current settings. Share this URL with others — when they open it, the settings are automatically applied and they can immediately begin processing images with your configuration.

Settings loaded from a URL take priority over locally saved settings.

19. Tips & Tricks

  • Social media banners — Use 16:9 aspect ratio with Overlay mode, Blur enabled, and a 3–5% margin for professional-looking cover images.
  • Thumbnail generation — Set a Max width of 300px, choose 1:1 (Square), and Crop mode with Center crop for consistent thumbnails.
  • Watermarking — Add a text overlay with low opacity (20–30%) and position it in a corner. Use a light color with an outline for visibility on any background.
  • Title cards — Use the text background feature with slant values to create angled banner effects behind your title text.
  • Consistent branding — Set up your preferred settings once, then use Copy Settings Link to save the URL as a bookmark. Use it every time for uniform results.
  • File size reduction — Force JPG output at 70–80% quality. Most images look identical but are significantly smaller.
  • Privacy — All processing happens in your browser. Images never leave your machine.
